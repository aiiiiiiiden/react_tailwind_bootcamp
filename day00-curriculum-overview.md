# React + Tailwind CSS v4 Bootcamp
## 14일 Active Learning 커리큘럼

**대상**: 경험 많은 개발자를 위한 실전 스타일링 집중 과정
**기간**: 14일, 매일 30분
**목표**: CSS 레이아웃과 Tailwind를 활용한 실전 컴포넌트 디자인
